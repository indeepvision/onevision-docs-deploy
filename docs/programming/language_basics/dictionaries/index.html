Dictionaries

Just like in TypeScript, dictionaries are objects used to store collections of key-value pairs. They allow you to define and manage data where the keys are not necessarily numerical indices, but rather unique identifiers like strings or particular numbers. This is useful for associating values with specific keys, much like a dictionary associates definitions with words.

Defining a Dictionary​

You can define a dictionary in OneVision using an object type where the keys are of a specific type (string or number) and the values can be of any type. You can easily define a dictionary with a key-value definition using the keyword Record. For example:

// Defining a dictionary with the keys being strings and the values arrays of numbers. 
var dict1: Record<string, number[]> = {}

You can index a dictionary just like with arrays: 

dict1["element1"] = [1,2,3,4];  
dict1["element2"] = [5];  
dict1["element25"] = [8,9,10];

Initializing dictionaries on creation​

Dictionaries can be created by initializing an object with key-value pairs.

var dict2: Record<number, string> = {
    1: "element1", 
    20: "element20"
}; 

dict2[22] = "element22";

Accessing and Modifying Dictionary Values​

You can access and modify dictionary values using the key. Adding new key-value pairs can be done by simply accessing an inexistent key.

var userAges: Record<string, number> = {
    "Winston Churchill": 90,
    "Anthony Eden": 79,
    "Harold Macmillan": 92
};

// Accessing values
userAges["Winston Churchill"]; // Outputs: 30

// Modifying values
userAges["Winston Churchill"] = 20;
userAges["Winston Churchill"]; // Outputs: 20

// Adding new key-value pairs
userAges["Alec Douglas-Home"] = 92;
userAges["Alec Douglas-Home"]; // Outputs: 92

// Deleting key-value pairs
userAges["Charlie"]; // Outputs: undefined

Creating static dictionaries​

In OneVision, you can also create static dictionaries. This can have any key-value pair and they can be accessed as properties of the object: 

var obj = {
    item1: 1, 
    item2: 22
}; 
var itemNumber = obj.item1; // itemNumber = 1

// Alternatively
var obj = {
    "item1": 1, 
    "item2": 22
}; 

var itemNumber = obj["item2"]; // itemNumber = 22

This statically typed objects are useful to create complex datatypes to share data between multiple sequences. Using static dictionaries, you can create objects with different types: 

var obj = {
    "item1": 1, 
    "item2": 22, 
    "item32": "asd"
}; 
var itemNumber = obj.item1; // itemNumber = 1
var itemString = obj.item32; // itemString = "asd"

// Correct modification
obj.item1 = 321; // Value of obj.item1 changed to 321

// Error changing type
// obj.item32 = 321; // Cannot cast number to string

Note the type is inferred by OneVision, so the dictionary definition cannot be changed.

Interfaces​

Interfaces in OneVision are a way to define a static shape of a dictionary object, describing what properties the object should have. They provide a way to enforce the structure of objects and ensure type safety. In other words, interfaces are a way to define the interface of static dictionaries. 

interface User{
    name: string; 
    age: number; 
}

var user: User = {
    name: "Pau", 
    age: 34
};

// Accessing user details
console.log(user.name); // Output: Pau
console.log(user.age); // Output: 34

// Modifying user details
user.name = "Jordi"; 
user.age = 35;

With interfaces you can define arrays of dictionaries enforcing certain structure. For example: 

interface User{
    name: string; 
    age: number; 
}

var users: User[] = []; 
users.push({
    name: "Pau", 
    age: 34
});
users.push({
    name: "Jordi", 
    age: 35
});

console.log(users[0].name); // Output: Pau 
console.log(users[1].name); // Output: Jordi

Iterating Over Dictionaries​

You can iterate over the keys of a dictionary using Object.keys().

var userScores: Dictionary<number> = {
    "player1": 100,
    "player2": 150,
    "player3": 200
};

// Using Object.keys()
var keys = Object.keys(userScores); 
var sumScores = 0; 
for(var i=0; i<keys.length; i++){
    sumScores = userScores[keys[i]]; 
} 

// Using Object.values()
var values = Object.values(userScores); 
var sumScores = 0; 
for(var i=0; i<values.length; i++){
    sumScores = values[i]; 
}
